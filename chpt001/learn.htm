The best way to learn is to learn the most basic unit of what you are learning and then once you've mastered that, move on to the next basic unit and learn that skill.

This is a kind of skill-stacking that feels slow (at first) but then builds in such a way that helps you so you don't get stuck later and helps you to not have to go back and re-learn things.

Methods That Help You Learn

  • Learn the basics. Make sure you understand everything about the foundational items of the thing you are learning.
  • Learn in small segments and test your knowledge. This means you should learn a small chunk, then try using that knowledge.
  • Push to new skills. Once you learn the underlying concepts, don't get stuck on those, but push to learn the next level skills.
  • Gain empirical understanding. Use the skills you learn and gain experience.

How Do These Apply To Make Games With JS?

This book breaks everything down into smalls pieces you can learn quickly.

This book takes the things you learn and uses it to build working apps -- even if they are very basic.
